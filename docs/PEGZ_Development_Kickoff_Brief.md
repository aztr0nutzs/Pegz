1. The Mission
We are building PEGZ, a mobile-first puzzle game that transforms the traditional "Peg Solitaire" mechanic into a high-contrast, edgy, and "crazy-looking" collectible experience. This is not a quiet board game; it is a visceral, loud, and mechanically deep reimagining.

2. Core Project Pillars
The Aesthetic: High-contrast, vibrant, and "gross-cool." Think neon-bio-labs, gothic-vampire-glamour, and urban-grit.

The Twist: Level-specific obstacles (Slick Surfaces, Contamination) and character-specific abilities (Shadow Jumps, Re-Gifting).

The Hook: A deep character collection system ("The Garage") where players collect sets of 8 unique PEGZ.

3. Immediate Technical Requirements (MVP)
The first milestone is a Functional Prototype containing:

Triangle Peg Logic: Standard jump-to-remove mechanics.

Theme 1 (Bio Lab): Implementation of "The Petri Dish" board and "Test Tube Terror" PEGZ.

Core Mechanic: The Contamination Zone (a target hole that locks a piece for one turn).

Audio/Visual: Initial "Squelch" feedback for successful jumps.
